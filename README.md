## Vibe Blog App

Проект создан для изучения подхода *vibe coding* в экосистеме Laravel и практики совместной разработки через Cursor.

### Цели
- Разобраться, как организовывать Laravel-приложения в стиле vibe coding.
- Освоить рабочий процесс в Cursor с использованием ИИ-ассистента.
- Отработать цикл разработки: постановка задач → реализация → тестирование.

### Используемые инструменты
- **Laravel 12** — основной фреймворк backend-части.
- **Inertia.js + Vue 3** — связка для фронтенда без разрыва SPA.
- **Tailwind CSS 4** — стилизация интерфейсов.
- **Cursor** — IDE с поддержкой AI для ускорения разработки.
- **Laravel Boost** — вспомогательные инструменты и интеграции для продуктивной работы.

### Быстрый старт
1. Склонируйте репозиторий и настройте окружение `.env`.
2. Установите PHP-зависимости (`composer install`) и JS-пакеты (`npm install`).
3. Выполните миграции и сидеры (генерируют тестовые данные).
4. Запустите локальный сервер Laravel и фронтенд-сборку.
